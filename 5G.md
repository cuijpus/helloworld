# <a id="tag-5G将带来哪些新变化" href="#tag-5G将带来哪些新变化">5G将带来哪些新变化</a>
2019 MWC:
5G绝对是重大亮点，华为、三星、一加、OPPO、小米等都展示出了自己的5G手机产品，比如华为Mate X、三星Galaxy S10 5G版、小米MIX 3 5G版等，但是5G网络的意义绝不仅仅是让消费者用上一部5G手机。很多人都对云端这个词比较熟悉，而5G网络则让云有了更具体的体现，一加的工程师展示了5G云游戏的使用，用户使用智能手机就能随时体验PC和主机端的大型游戏，并且不需要下载就能获得超高的画质和低延迟，之前许多用户也体验过云电脑这样的应用，但是受限于4G网络的速度和延迟，所以没有普及开来，但是5G网络的天然优势，可以完美客服这个难题。

# 5G 改变手机形态
内存会一直加下去？不！<br>
现在增加RAM：2GB->3GB->4GB->6GB->8GB->12GB（厂家成本越来越大）<br>
在5G网络下，对RAM的要求就会小了。<br>
反过来演进：12GB->8GB->6GB->4GB->2GB <br>
云端就把大量的计算给解决掉了，终端就显示一下而已！<br>
5G将使移动终端实现云计算化提供可能，很有可能你的手机将变成一块单纯的屏幕，计算和存储都将放在网络那头的服务器，到时候，你随便拿一个终端屏幕登陆服务器，都将是你的手机。

手机形态的进化，越来越摸不透了
说完5G，我再总结下今年的终端设备。今年的终端设备进化路数跟去年的“全面屏2.0”还很不一样。折叠手机和柔性屏彻底打消了手机党和平板党之间的顾虑与疑惑。而“腕表手机”在我眼里本身就是一款可能“革功能机的命”的产品，这些产品的出现，无疑让被指责为“亦步亦趋”的手机市场平添了一份亮色。

不过讲真，在5G网络补足终端设备性能的趋势下，未来可能手机的形态做工也将不再像今日这般重要，可能只是5G服务的一个承载体。这也就是我上文起这样一个小标题的原因——5G使得手机形态的进化路线更加多样了。

# 5G 云游戏
高画质，无卡顿！<br>
5G时代带给我们最直观和最明显的影响就是高速率。在4G网络环境下，下载一部1G的电影估计要30S，而用5G网络，则不会按秒来计算。据说5G速率已达到4G的100倍，甚至可能达到上万倍。到时，玩游戏、看视频几乎将不会再出现“卡”的情况。

2019-02-27[腾讯云游戏平台“腾讯即玩”公布 适用于PC、手机](https://wap.gamersky.com/news/Content-1157575.html)<br>
MWC大会上腾讯与英特尔联手，推出云游戏平台“腾讯即玩”。“腾讯即玩”借助云服务能力，在云端完成最耗费硬件资源和功能，从而让玩家摆脱硬件和平台的束缚，省去漫长的下载和等待时间，在不同联网的终端上都可以获得高品质的游戏大作体验。“腾讯即玩”将适用于PC和智能手机，将在下个月的GDC上公布更多细节

一加展台的5G云游戏:
当然了，鉴于我在展台上也切身体会到了5G云游戏的魔力，因此对于虎哥的“5G网络将取代大内存手机”的看法深表赞同。这极有可能成为手机未来的进步方向，将智能手机的“生活中枢”作用发挥的更加淋漓尽致。

# 5G AR/VR 云化
5G的高宽带允许和低延迟，带来了大量数据的传输。解决了VR因网络带宽不足、网络延迟高导致的视频卡顿，画面质量差等情况。这将使得我们的虚拟现实体验更加真实，观影和游戏更加身临其境。

微软在MWC2019上重点介绍智能云、Azure Kinect边缘计算设备，并且发布了全新的硬件，也就是第二代AR头盔 HoloLens 2，国内用户可能对HoloLens这款产品比较陌生，它是一款AR设备，没有线缆和听筒，并且不需要连接电脑，具有全息、高清镜头、立体声等特点，可以让你看到和听到你周围的全息景象。目前HoloLens主要还是用于工业领域，比如在检查涂层厚度方面，过去需要两个人花一天时间才能完成，但配合使用HoloLens之后，一个人在2小时内就能完成同样的工作，不过随着5G时代的到来，这种设备将很快走进千家万户。

# 5G 无人驾驶
5G还带来了许多实质性的突破。如无人驾驶，5G网络的延时由4G的30-50毫秒降低到了1毫秒，反应速度加快了，将使得无人驾驶将更加安全。
因为自动驾驶有一个快速搜集信息，快速进行人工智能决策，快速反映机制，如果速度慢了是很难做到的。
